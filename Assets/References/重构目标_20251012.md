# VRBrush 重构目标文档
**日期**: 2025年10月12日  
**目标**: 实现ADBrush（Adaptive Drawing Brush）及相关系统重构

---

## 📋 项目背景

### 架构澄清
- **Ray** 实际上是 **Axis**（轴线可视化）
- **Plate** 实际上是 **Disk**（圆盘可视化）
- 当前架构图中的命名已经对齐

### 新功能目标
创建一个新的笔刷系统 **ADBrush**，结合以下研究领域的优势：
1. **VR空间绘画** (SurfaceBrush 2021, AdaptiBrush 2021)
   - 优点：流畅的3D轨迹控制
   - 缺点：笔刷形状固定（线段），rolling控制不连续
   
2. **Sweep-based建模** (3-Sweep 2013, Blender Sweeping)
   - 优点：通过2D截面扫掠生成3D形状
   - 缺点：难以通过简单直观的交互创建复杂形状

### 核心创新点
- **可变形截面 (Morphable Cross-Section)**: 绘制过程中截面可以动态变形
- **点按式绘制**: 多次拖拽操作，使用Catmull-Rom样条连接
- **实时Morph控制**: 通过摇杆和UI实时调整形状变形
- **后处理能力**: 绘制完成后可编辑每个关键点

## 🎯 核心概念：3DMM (3D Morphable Model) 风格

### Morph变形公式
```
p = (o + Σλᵢ·Δᵢ) × size

其中:
- o: 基础形状顶点（笛卡尔坐标）
- Δᵢ: 第i个morph的变形向量（笛卡尔差值）
- λᵢ: 第i个morph的系数 [0, 1]
- size: 整体缩放系数（特殊内置morph）
```

### Morph系数规则
- **普通morph**: 默认 λ = 0，范围 [0, 1]
- **Size (特殊morph)**: 默认 λ = 1，范围 [0, 1]，乘法应用（先加后乘）
- **未变化的顶点**: Δ = (0, 0)

### 关键设计决策
**✅ Morph作用于BrushShape，而非StrokePoint**
- StrokePoint只负责路径控制点
- BrushShape定义2D截面形状
- Morph在截面层面进行变形

## 🏗️ 数据结构重构

### Phase 0: 类重构（删除冗余继承）

#### 问题
- `RibbonStroke` 是空壳基类，只有一个子类 `ShapeStroke`
- 继承层次无意义，增加复杂度
- 命名暗示"丝带状"，但实际支持任意截面形状

#### 解决方案
```
删除: RibbonStroke.cs（无用基类）
重命名: ShapeStroke → Stroke（唯一实现）
重命名: RibbonPoint → StrokePoint（更准确的命名）

新结构:
- StrokePoint.cs  ← 笔画控制点
- Stroke.cs       ← 笔画类（包含所有mesh生成逻辑）
- BrushShape.cs   ← 2D截面形状定义
```

### Phase 1: 坐标系统转换（极坐标 → 笛卡尔）

#### 问题
- 极坐标在morph计算时存在角度周期性问题
- 例如: θ = 0.9π 到 θ = -0.9π，差值应该是 0.2π（短路）而非 -1.8π（长路）
- 多个morph叠加时，极坐标差值计算复杂且容易出错

#### 解决方案：全面转向笛卡尔坐标

**BrushShape 内部存储**:
```csharp

    public Vector2 GetNodePolar(int index);
    public void SetNodePolar(int index, Vector2 polar);
}
```

**JSON 格式**:

```json
  "name": "triangle",

    {"x": 0.5, "y": 0},
```

### Phase 2: Morph系统数据结构

#### MorphVector（变形向量）
```csharp
[Serializable]
public class MorphVector
{
    public string name;                          // morph名称（如"wider", "rounder"）
    public List<Vector2> deltaVertices;          // 笛卡尔差值 Δ = n - o
    public float minValue = 0f;                  // 最小系数
    public float maxValue = 1f;                  // 最大系数
    public float defaultValue = 0f;              // 默认系数（size例外）
    
    // 创建morph向量
    public static MorphVector Create(BrushShape origin, BrushShape target, string name)
    {
        var morph = new MorphVector { name = name };
        for (int i = 0; i < origin.NodeCount; i++)
        {
            Vector2 delta = target.Nodes[i] - origin.Nodes[i];  // 笛卡尔差值
            morph.deltaVertices.Add(delta);
        }
        return morph;
    }
}
```

#### BrushShape 扩展
```csharp
public class BrushShape
{
    // 现有字段...
    [SerializeField] private List<MorphVector> morphs = new List<MorphVector>();
    
    public IReadOnlyList<MorphVector> Morphs => morphs.AsReadOnly();
    
    // 添加morph
    public void AddMorph(string name, BrushShape targetShape);
    
    // 应用多个morph得到新形状
    public BrushShape ApplyMorphs(Dictionary<string, float> morphValues)
    {
        var result = new BrushShape(this.Name);
        
        for (int i = 0; i < NodeCount; i++)
        {
            Vector2 pos = nodes[i];  // 基础位置
            
            // 应用所有普通morph（加法）
            foreach (var (morphName, lambda) in morphValues)
            {
                if (morphName == "size") continue;
                var morph = GetMorph(morphName);
                pos += lambda * morph.deltaVertices[i];
            }
            
            // 应用size（乘法，后处理）
            if (morphValues.TryGetValue("size", out float size))
            {
                pos *= size;
            }
            
            result.AddNode(pos);
        }
        
        // 复制边
        foreach (var edge in Edges)
            result.AddEdge(edge);
            
        return result;
    }
}
```

```csharp
public struct StrokePoint
```

## 🎨 界面实现

### Phase 3: MorphEditor（创建Morph）

#### 功能
- 为已有形状创建morph变形向量
- **不是**扩展BrushEditor，而是**新建**独立的MorphEditorController

#### 交互方式
```
1. 加载基础形状 o (例如 triangle.json)
2. 在左手极坐标平面上显示该形状（UI层仍用极坐标）
3. 用户拖拽圆盘上的顶点 → 实时预览变形
4. 完成后，系统计算 Δ = n - o（笛卡尔差值）
5. 自动命名为 morph1, morph2, morph3...
6. 保存：将morph追加到JSON的morphs数组
```

#### 软绳拖拽机制
```
投影点 ●────────● 选中的顶点
       ↑软绳
       长度d

if (d > threshold)  // 例如 0.05
{
    // 顶点柔和地跟随投影点
    vertex = Lerp(vertex, projectedPoint, dragSpeed * deltaTime);
}

特点:
- 超过阈值才拖动（避免抖动）
- 柔和跟随（不是瞬移）
- 实时预览变形效果
```

#### 自动命名逻辑
```csharp
string GetNextMorphName(BrushShape shape)
{
    int maxNum = 0;
    foreach (var morph in shape.Morphs)
    {
        // 匹配 "morph1", "morph2" 等
        var match = Regex.Match(morph.name, @"morph(\d+)");
        if (match.Success)
        {
            int num = int.Parse(match.Groups[1].Value);
            if (num > maxNum) maxNum = num;
        }
    }
    return $"morph{maxNum + 1}";
}
```

### Phase 4: ADBrushController（使用Morph）

#### 绘制模式：点按式
```
旧模式 (AdaptiBrush/HelioBrush):
- 持续按住扳机 → 连续采样点

新模式 (ADBrush):
- 点击扳机 → 创建控制点1
- 松开
- 调整morph（可选）
- 再次点击 → 创建控制点2
- ...
- 系统用Catmull-Rom样条连接所有控制点
```

#### UI交互（修正版）

**左摇杆 - 线性列表选择morph**:
```
不是径向菜单！而是上下滚动列表：

左摇杆上推 → 高亮上一个morph
左摇杆下推 → 高亮下一个morph

UI显示:
┌──────────────┐
│   morph1     │
│ ► morph2 ◄   │ ← 当前高亮（带箭头指示）
│   morph3     │
│   size       │
└──────────────┘

实现:
- 循环滚动（到底后回到顶部）
- 防抖处理（避免连续触发）
```

**右摇杆 - 圆形滑块控制morph值**:
```
         12点 (0°)
         λ = 0
           ↑
           │
 9点 ←─────●─────→ 3点
 λ=0.75    │     λ=0.25
           │
           ↓
         6点 (180°)
         λ = 0.5

公式: λ = angle / 360°

特点:
- 顺时针方向递增
- 12点方向为0
- 转一圈回到12点为1.0
- 推得越远，精度越高（死区处理）
```

**代码实现**:
```csharp
float GetMorphValueFromJoystick(Vector2 rightStick)
{
    if (rightStick.magnitude < 0.1f) 
        return currentMorphValue;  // 死区
    
    // 计算角度 (0° = 12点 = Vector2.up)
    float angle = Mathf.Atan2(rightStick.x, rightStick.y) * Mathf.Rad2Deg;
    
    // 转换为 [0, 360)
    if (angle < 0) angle += 360f;
    
    // 转换为 [0, 1]
    float lambda = angle / 360f;
    
    return Mathf.Clamp01(lambda);
}
```

```csharp
// Size是内置morph，自动添加
public void EnsureSizeMorph()
{
    if (!HasMorph("size"))
    {
        var sizeMorph = new MorphVector
        {
            name = "size",
            deltaVertices = new List<Vector2>(),  // 空的，因为是乘法
            defaultValue = 1f,
            minValue = 0f,
            maxValue = 2f,
            isMultiplicative = true  // 标记为乘法类型
        };
        morphs.Add(sizeMorph);
    }
}
```
